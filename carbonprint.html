About

The Rasterizer is a group project with me and Axel Weinz. The scene is built from scratch in C++ and the challange for us was to create a Clipping and Shadow mapping algorithm.
The simplest technique for creating shadows in a computer graphics scene is by implementing a raytracer that detects collision with an object and creates a shadow based on that for every frame.
Real-time ray tracing is very computation heavy and thus Shadow mapping is used as an alternative, shadow mapping is more difficult to code but is much less computation heavy.

In a shadowmapping algorithm, shadows are created by testing whether a pixel is visible from the light source, by comparing the pixel to a z-buffer or depth image of the light source's view.

To make the shadowmapping algorithm we also had to implement a clipping algorithm so that the application doesnt crash when the camera is out of bound, this is needed since the light source view in most cases are out of bound.
