About

Plack was created in the course 'Mobile development with web technologies'.
Its a forum with an interface inspired by the application 'Jodel'. In plack, students can ask and answer questions anonymously.

The app also has google maps integration where users can create study groups that other users can see on the map

The app was created with Angular.js and Apache Cordova and implements Google Maps API
